Instructions on directories
---------------------------

In the 'models' folder are the provided models for Glycolysis, Pentose Phosphate Pathway, AMP Salvage Network, and the Hemoglobin module. These are the model files you must load and use.

Place finalized RBC construction notebook into directory `notebooks`
Place finalized enzyme reconstruction notebooks in directory `notebooks`

Export reconstructied RBC to "models" folder as JSON
Export reconstructied enzymes to "models" folder as JSON

Place finalized simulation case study notebooks in directory `case_studies`
Export figures generated in the simulation case study notebooks as PDFs or PNGs into the directory `figures`

For exported figures that will be used as a main result in the report, copy figure PDFs or PNG files and place the newly made copies into the directory `report`. Exported figures MUST have the same name as their Figure number. For example, the first figure in the report "Figure. 1" would be a file called "figure1.pdf"
